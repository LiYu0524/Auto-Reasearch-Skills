# Auto Research Skills

Claude Code 一站式学术研究工作流技能集，覆盖从论文检索到最终演示的完整链路。

## 包含的子技能

- **paper-research** - arXiv 论文检索、PDF 提取、文献综述生成
- **paper-reviewer** - 论文精读、方法讲解、审稿式优缺点分析
- **zotero** - Zotero 文献库管理（通过 MCP）
- **google-docs** - Google Docs 文档读写、Drive 文件管理
- **paper-banana** - 学术插图生成（AI 驱动，Planner → Visualizer → Critic 循环）
- **drawio** - draw.io 架构图、流程图生成
- **pptx** - PowerPoint 演示文稿生成

```bash
git clone git@github.com:LiYu0524/Auto-Reasearch-Skills.git ~/.claude/skills/auto-research
```
